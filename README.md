# Gate AI Quant Professional 9.2

本版本是 v14 路线的第一阶段：交易决策中心。

## 核心升级

- 扫描结果改为移动端交易卡片。
- 每张卡显示方向、模型信心、盈亏比、多空评分和推荐等级。
- 增加“AI分析”“K线数据”“模拟交易”“收藏”四个按钮。
- 点击“模拟交易”会直接带入币种、方向、入场、止损和止盈，无需重新输入。
- 收藏保存在当前浏览器本地。

## 重要说明

模型信心是规则评分，不等于可保证的真实胜率。模拟交易不连接 Gate 账户，也不会发送真实订单。

Render 免费实例重新部署可能清空本地 SQLite 数据。需要长期保存时，请配置持久化磁盘并设置：

`SIM_DB_PATH=/var/data/sim_trades.db`
