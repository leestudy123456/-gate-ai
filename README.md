# Gate AI Quant Professional 9.0

移动端量化研究、概率校准与模拟交易日志工具。读取 Gate 公开合约行情，不需要 API Key，不连接交易账户，也不会自动下单。

## 9.0 模拟交易
1. 在“研究”中运行 AI 决策。
2. 点击“带入模拟交易”。
3. 核对方向、入场、止损、止盈、余额、风险比例和杠杆。
4. 选择市价模拟或策略触价入场后建立订单。
5. 在“模拟”页点击“更新行情”，系统使用已收盘K线检查成交、止盈、止损和超时。

模拟结果计入双边手续费与滑点。同一根K线同时触发止盈和止损时，由于OHLC不能确定盘中先后顺序，系统保守按止损先发生。

## 数据保存
默认 SQLite 文件：`data/sim_trades.db`。

Render 免费实例的本地文件可能在重新部署或实例重建后消失。长期保存时应挂载 Render Persistent Disk，并设置：

`SIM_DB_PATH=/var/data/sim_trades.db`
