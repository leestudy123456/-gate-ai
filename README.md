# Gate AI Quant V5

Gate公开行情研究、策略验证、市场扫描和预测价值计算平台。

## V5模块

### 1. 实时分析
- 5m / 15m / 30m / 1h / 4h
- 只使用已收盘K线
- EMA、MACD、RSI、ATR、成交量、价格结构
- 多空评分、参考入场、止损和目标

### 2. 策略研究
- 无前视回测
- 手续费和滑点
- 参数优化
- Walk-Forward样本外验证
- Monte Carlo风险分析

### 3. Gate市场扫描
- 从USDT永续合约行情中按24小时流动性筛选
- 最多扫描30个合约
- 并发受限，避免高频请求
- 排行榜显示方向、评分和流动性
- 五周期加权共振

### 4. 信号日志
- 每次手动实时分析写入SQLite
- 可通过 `/api/signals` 查看最近记录
- Render免费实例本地磁盘可能在重启后清空，后续需要持久磁盘或外部数据库

### 5. 预测市场价值
- 手动输入YES市场价格
- 手动输入独立估计的真实概率
- 计算概率优势、期望值、期望收益率和封顶Kelly
- 不会自动生成“真实概率”
- 当前没有使用未经确认的Gate Prediction私有或非公开接口

## 不包含
- 自动下单
- 账户API密钥
- 资金转移
- 自动生成新闻事件概率
- 盈利保证

## 研究门槛
- 样本外交易数至少50
- Profit Factor建议高于1.20
- 单笔期望为正
- 最大回撤建议低于15%
- 至少2至4周模拟盘
