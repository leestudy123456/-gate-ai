# Gate AI Quant Professional 4.0.0

面向手机端的公开行情量化研究平台。它不连接Gate账户，也不会自动下单。

## 核心能力

- 已收盘K线实时分析与数据质量检查
- 5m / 15m / 1h / 4h 多周期共振
- 方向、参考入场、止损、目标、盈亏比和失效条件
- 历史同方向命中率、成本后有效率、样本量和95%置信区间
- 无未来数据的策略回测
- 参数优化、Walk-forward滚动样本外验证、Monte Carlo路径风险
- 市场扫描、信号日志、仓位预算和预测市场价值计算
- 专业术语点击解释与模型说明卡

## 概率说明

页面中的“模型评分”不是胜率。只有运行历史方向验证后，系统才显示历史校准命中率；该结果仍不是未来收益保证。

## Render部署

- Python: `3.12.11`
- Build: `pip install -r requirements.txt`
- Start: `uvicorn app:app --host 0.0.0.0 --port $PORT`
- Health: `/api/health`

## 验收

部署后依次检查：

1. `/api/health` 返回 `4.0.0`
2. `/api/model-card` 正常返回模型方法和限制
3. 首页“总览/研究/排行/预测”可以切换
4. 研究页可运行实时分析、方向验证、回测和策略计划
