# Gate AI Quant V12.1 Stability Edition

移动端优先的 FastAPI 量化研究与模拟交易工具。只读取 Gate 公开行情，不连接账户、不发送真实订单。

## V12 核心升级

- AI 决策与 K 线分析改为两个独立接口，避免两个按钮显示相同结果。
- 新增快速策略接口 `/api/strategy/quick`，使用最近已收盘 K 线生成方向、入场、止损、止盈和仓位建议，不再把耗时历史验证塞进同一次请求。
- 仓位管理器加入 LONG/SHORT 方向校验、双边手续费、滑点、保证金上限、杠杆容量、最坏预计亏损、安全评分和参数警告。
- 保留 V11.3 的自定义最长持有 K 线、AI 动态退出、保本、最小亏损、移动止盈和观察窗口。
- 前端超时提示会显示实际等待秒数，不再把所有错误统一写成“超过30秒”。
- 扫描页的“AI分析”和“K线数据”现在分别调用不同功能。

## 部署

1. 将 ZIP 解压后的全部文件上传到 GitHub 仓库根目录。
2. Render 使用仓库内的 `render.yaml` 自动部署。
3. 健康接口应显示版本 `12.1.0`。
4. 页面顶部应显示 `V12 PROFESSIONAL`。

## 数据目录

`data/.gitkeep` 用于确保 Git 保留数据目录。SQLite 文件会在运行时自动创建。Render 免费实例的本地磁盘可能在重启后清空；长期保存需配置持久磁盘，并设置 `SIM_DB_PATH` 和 `SIGNAL_DB_PATH`。

## 风险提示

所有评分、概率、回测和仓位结果仅用于研究。模型评分不等于真实胜率，历史结果不保证未来表现。


## V12.1 稳定性修复

- 修复模拟交易在没有新增已收盘 K 线时 `note` 未初始化导致的刷新失败。
- 修复 `requested` 在初始化 `best_price` 后才赋值的潜在错误。
- 单笔模拟交易刷新失败时隔离错误，其余交易继续更新。
- 刷新错误现在显示交易方向、状态和 Python 异常类型，便于手机端定位。
