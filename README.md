# Gate AI Quant V13 Professional

本版本重点修复排行榜扫描超时与服务状态不透明：

- 12个合约并发扫描（并发上限8）
- 30秒扫描结果缓存，Gate短暂超时时可返回5分钟内最近成功结果
- 后端单次扫描12秒保护，避免请求无限卡住
- 前端等待上限45秒，兼容Render休眠后的冷启动
- 服务状态检查与手动重试
- 可选每30秒自动刷新排行榜

部署入口：`uvicorn app:app --host 0.0.0.0 --port $PORT --workers 1`

健康检查：`/api/health`，应返回版本 `13.0.0`。

本工具只使用公开行情，不连接账户、不自动下单，不保证盈利。
