# Gate AI Quant Professional 11.1 Stable

手机优先的公开行情研究平台。V11 在 V10 的样本外校准、风险引擎和模拟交易基础上加入：

- 动态 EMA：5m 使用 9/21/55/200，15m/30m 使用 20/50/100/200，1h/4h 使用 20/50/200。
- 多因子投票：趋势、动量、成交量、波动率、衍生品；链上和订单流采用可用性门控，未接入时保持中性。
- 衍生品分析：资金费率、资金费率动量和 OI 快照。
- 动态市场状态权重、策略质量等级 S/A/B/C/D。
- AI Strategy Lab：模拟交易日志、策略等级/市场状态归因、最大回撤、连胜连亏。
- 决策回放：读取创建交易时保存的模型、概率、风险和 Trade DNA 快照。

## 运行

```bash
pip install -r requirements.txt
uvicorn app:app --host 0.0.0.0 --port 8000
```

模拟交易仅写入本地 SQLite，不连接账户，不发送真实订单。链上数据与订单流在没有可靠数据源时不会伪造信号。


## 11.1 稳定性说明

- 启动时自动创建 `data/`。
- 自动初始化信号数据库和模拟交易数据库。
- `data/.gitkeep` 用于让 GitHub 保留目录。
- Render 免费实例的本地磁盘不是永久存储；重新部署后模拟记录可能丢失。长期保存需配置持久磁盘，并设置 `SIM_DB_PATH` 与 `SIGNAL_DB_PATH`。
